# NoAIOverview
Browser Extension to remove AI Overview content.<br>
Made by Chroma from https://haxs.dev/.<br>
This extension removes AI Overview content.<br>

Ironically, the code was made with the help of AI itself.<br>
The code generated with AI was generated following SAID methods (please read https://stopai.haxs.dev/ or visit https://haxs.dev/artificialintelligence)<br>

# How to Install
Click on "Code", click on "Download ZIP".<br>
<img width="919" height="316" alt="tutoria" src="https://github.com/user-attachments/assets/84412c97-e6cc-4aeb-a1f6-85ef74113971" /><br>
Extract the folder on your computer.<br>
Save the extracted files somewhere.<br>

Now, open a Chromium based browser, like Google Chrome or Opera.<br>
Go to the extension page (enter chrome://extensions/ in the search bar on top), and click "Enable developer mode"<br>
<img width="1920" height="239" alt="extensions" src="https://github.com/user-attachments/assets/124729e1-19d0-40d9-846c-8436fb6116af" /><br>
Next, click on Load unpacked.<br>
<img width="392" height="117" alt="image" src="https://github.com/user-attachments/assets/9c19f991-338b-400c-b8ee-96c4768a91a8" /><br>
Select the UNZIPPED FOLDER (not the zip file) from your computer.<br>
